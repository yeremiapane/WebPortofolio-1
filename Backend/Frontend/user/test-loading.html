🔧 Cara Kerja Loading Screen

📋 Proses Loading:

  1. Initialization - Loading screen muncul saat halaman dimuat
  2. Progress Simulation - Progress bar bergerak dengan increment random
  3. Message Updates - Pesan berubah berdasarkan progress:
    • Loading Portfolio/Project Details...
    • Preparing Content...
    • Loading Images...
    • Initializing Components...
    • Almost Ready...
    • Ready!
  4. Resource Loading - Menunggu semua resource (gambar, CSS, JS) selesai dimuat
  5. Minimum Time - Loading minimal 1 detik untuk UX yang baik
  6. Smooth Exit - Fade out dengan animasi dan trigger entrance animations

⚡ Performance Features

🚀 Optimizations:

  • Prevent Scrolling - Mencegah scroll saat loading
  • DOM Cleanup - Menghapus loading screen dari DOM setelah selesai
  • Event Listeners - Proper cleanup untuk mencegah memory leaks
  • Smooth Animations - Hardware-accelerated CSS animations
  • Progressive Enhancement - Fallback jika JavaScript disabled
